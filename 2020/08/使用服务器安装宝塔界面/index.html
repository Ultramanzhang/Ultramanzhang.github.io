使用服务器安装宝塔界面

使用服务器安装宝塔界面

本教程基于阿里云编写。

准备工作

  • 购买一台云服务器
  • 安装宝塔界面

购买云服务器

参见阿里云官方教程开通云服务器

安装宝塔界面

一、首先进入宝塔界面官方网站

然后我们找到Linux版本

这里我的服务器是使用的 CentOS 7.8 64位操作系统。

  • 找到这个界面
  • 点击查看后，我们复制第一条命令

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh
  • 复制完毕后我们来到服务器ssh终端，这里我使用的是xshell链接 输入以上命令后回车安装

    输入y，并回车。接下来便是等待宝塔面板进行安装。

    这是宝塔面板的初始账号密码，一定要记住，只会生成一次。

我们得到登录宝塔面板的URL,账号和密码。

面板地址：http://{您的服务器IP}:8888

初始化：第一次使用会要求初始化配置，请按照提示填入配置信息

在浏览器中输入 http://{您的服务器IP}:8888进行访问你的面板 如果找不到你的面板可以在xshell中输入一以下命令进行查找

/etc/init.d/bt default

这里会列出你的登录URL和用户名，密码是加密处理过的

  • 这样就是登录成功了

设置宝塔面板

  • 一定要把默认端口换掉

    • 首先进入面板设置

    • 在这里可以设置端口等属性

设置面板静态加速

  • 首先我们在软件商城中搜索 堡塔面板静态文件加速
  • 等待自动安装
  • 安装完毕后我们可以在主页找到
    • 点击进入，找到延迟最小的节点进行加速

  到此面板已经大致安装完毕，其余的功能自行探索就OK，或者参考这篇教程宝塔面板 安装与使用教程
