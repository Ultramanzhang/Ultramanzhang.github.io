java变量配置

在我们学习的过程中我们需要很多的变量，而在学习java的路上我们首先需要搭建java虚拟机的运行环境，在这里我还是选择装jdk1.8。毕竟对于大多数人来说JDK1.8是绝对够用的

  • 环境变量的配置 我们进入到jdk安装目录（C:\Program Files\Java\jdk1.8.0_131）可以看到以下内容
    • bin目录是存储一些可执行的二进制文件；

配置环境变量

环境变量是在操作系统中一个具有特定名字的对象，它包含了一个或者多个应用程序所将使用到的信息。
Path是一个常见的环境变量，它告诉操作系统，当要求系统运行一个程序而没有告诉它程序所在的完整路径时，系统除了在当前目录下寻找此程序外，还应到哪些目录下寻找。

  • 设置path环境变量
    1.右键计算机——>属性——>高级系统设置，出现图所示界面。

2.单击环境变量按钮，进入图所示环境变量操作主界面：

3.单击新建按钮，新建JAVA_HOME变量（用于说明JDK的安装目录），如图所示 这里你也可以浏览目录

4.保存后，修改环境变量path将下面字符添加到path中

%JAVA_HOME%\bin

5.查看java是否安装成功

  • 按住win+R输入cmd

  • 键入以下命令
java -version


出现这个界面说明安装成功

  • 或者输入下面这个命令
javac


出现这个界面也是安装成功
