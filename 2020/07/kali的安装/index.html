安装VMware Tools

  • 点击打开终端
  • 输入这个命令获取最新安装包
apt update
  • 输入这个命令安装VMware Tools
apt install open-vm-tools-desktop fuse
  • 输入y
  • 这里我输入的是Y

  • 安装完毕后输入这个命令重新启动
reboot
